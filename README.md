# 達源技術有限公司 透鏡成像產生器

完整功能版：
- 參數設定
- XZ 空氣透鏡圖
- XY 出光面圖
- 彩色等照度圖
- 中心線平台折線
- STEP/STL 輸出

GitHub Actions：
1. 上傳 LensDesignerPro.py / README.md / .github/workflows/build-exe.yml
2. 根目錄可放 icon.ico
3. Actions -> Build LensDesigner Pro EXE -> Run workflow
4. 下載 LensDesignerPro-Windows-Folder
5. 解壓後保留整個資料夾，執行 LensDesignerPro.exe

注意：CadQuery 必須用 onedir 打包，不要只複製單一 EXE。
